# SOP 010: Study Retirement

## Status: ACTIVE | Version: 1.0

**Purpose**: Retire studies that are no longer relevant or have been superseded.

---

## When to Retire

| Condition | Action |
|---|---|
| Null confirmed + replicated | Retire to archive |
| Superseded by new study | Retire + link new |
| methodology flaw | Retire + note flaw |
| Out of scope | Archive for reference |

---

## Procedure

### 1. EVALUATE

```
□ Review all results
□ Check replication status
□ Check relevance
□ Check quality
```

### 2. DECIDE

```
□ Status: archived | superseded | retracted
□ Reason: document specific reason
□ Replacement: link if superseded
```

### 3. DOCUMENT

```
□ Final report with retirement note
□ Update in experiment tracker
□ Update in vacuum detector
□ Notify team
```

### 4. ARCHIVE

```
□ Move to archive folder
□ Update index
□ Unlist from active studies
```

---

## Quality Gate

| Check | Requirement |
|---|---|
| Complete review | All results checked |
| Clear reason | Documented |
| Links maintained | Superseded linked |
