# Research Quests — Gamification System

## Purpose

Make research feel like an adventure, not a chore.

---

## Quest Types

### Main Quests (Required)

| Quest | Description | XP | Unlocked By |
|-------|------------|-----|------------|
| **First Study** | Complete a registered study | New researcher |
| **Vacuum Hunter** | Detect 10 vacuums | Research Atlas |
| **Replication Hero** | Replicate a study | First study |
| **Null Reporter** | Report null findings | Any study |

### Side Quests (Optional)

| Quest | Description | XP | Trigger |
|-------|------------|-----|---------|
| **Speed Reader** | Complete reading ladder | Time |
| **Data Diver** | Find new evidence | Manual |
| **Code Ninja** | Fix analysis bug | GitHub |
| **Peer Reviewer** | Review 3 studies | Assignment |
| **Synthesizer** | Cross-pillar meta-analysis | Multiple studies |

### PvP Quests (Competition)

| Quest | Description | XP | Opponent |
|-------|------------|-----|---------|
| **Fastest Hypothesis** | First to pre-register | Others |
| **Biggest Effect** | Most significant result | Others |
| **Most Cited** | Most references | Others |

## XP System

### Earning XP

| Action | XP |
|---|---|
| Pre-register study | +50 |
| Complete primary analysis | +100 |
| Publish null finding | +75 |
| Replicate study | +150 |
| Cross-pillar synthesis | +200 |
| Mentor peer | +50 |

### Losing XP

| Action | XP |
|---|---|
| P-hacking discovered | -200 |
| Unregistered analysis | -100 |
| Missing pre-registration | -50 |

### Levels

| Level | Title | XP Required | Bonus |
|-------|-------|-------------|-------|
| 1 | Novice | 0 | — |
